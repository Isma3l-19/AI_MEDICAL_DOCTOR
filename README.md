# AI_MEDICAL_DOCTOR
An AI that instantly predicts a possible medical condition of a patient.

## Project Overview
The AI_MEDICAL_DOCTOR project is designed to leverage artificial intelligence to assist in the preliminary diagnosis of medical conditions based on patient symptoms. 

## Features
- Instant prediction of possible medical conditions.
- User-friendly interface for inputting symptoms.
- Machine learning algorithms trained on a diverse dataset of medical cases.

## Installation
To set up the project locally, follow these steps:
1. Clone the repository:
    ```bash
    git clone https://github.com/YourUsername/AI_MEDICAL_DOCTOR.git
    ```
2. Navigate to the project directory:
    ```bash
    cd AI_MEDICAL_DOCTOR
    ```
3. Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage
To run the application, execute the following command:
```bash
python app.py
```
Follow the on-screen instructions to input symptoms and receive predictions.
